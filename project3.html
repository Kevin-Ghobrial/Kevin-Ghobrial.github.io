Color Finding Bot

Description

This program can find the top color in any given image array. I did this using assembly. I tried to keep my program as efficient as possible by minimizing the static code I used. I also optimized my code to lower my dynamic instruction count. To do this, I used strategies such as optimizing what does or doesn't go into for-loops. This allowed me to bring my dynamic instructions down to 6000 and my static down to 54.
